Getting Started

Caduceus runs inside Hermes Agent. If you don't have Hermes yet, install it first.

Prerequisites

  • Hermes Agent — Install from GitHub
  • Python 3.11+ — For the SQLite database
  • macOS or Linux — Terminal-only for now (no Windows support yet)

Install

One command. Downloads and installs all skills into ~/.hermes/skills/.

curl -fsSL https://get.caduceus.sh | sh

That's it. No server to start. No account to create. The skills are installed and ready to use.

Load the orchestrator

Restart Hermes or start a new session, then load the orchestrator skill:

hermes chat
/ skills caduceus-orchestrator

The orchestrator is the foreman — it coordinates all the other agents.

Bootstrap a project

Themis runs a structured interview to set up your project. It asks six questions, then creates:

  • SPEC.md — What you're building and why
  • context.md — Current state and team
  • initial-tasks.md — First tasks to tackle
  • SQLite task entries — Ready for agents to claim
"bootstrap a new project with Themis"

Run your first task

Tell the orchestrator what you need:

"run the engineer on build-landing-page"
"run the researcher on competitor analysis"
"show me pending tasks"

The orchestrator spawns the right agent, tracks the task, and reports back when it's done.

What just happened?

Here's the full flow:

  1. You — tell the orchestrator what you need
  2. Orchestrator — creates a task in SQLite, selects the right agent
  3. Agent — spawned by Hermes, reads SPEC.md from QMD, does the work
  4. Agent — writes output to QMD, updates coordination log
  5. Orchestrator — marks task complete, reports to you

All without you in the loop — after the initial request.

Uninstall

To remove Caduceus:

rm -rf ~/.hermes/skills/caduceus-*
rm -rf ~/.hermes/caduceus
rm -f ~/.hermes/caduceus.db
