FAQ

What is Caduceus?

Caduceus is a framework for orchestrating teams of specialized AI agents that run inside Hermes Agent. It provides a collection of skills (orchestrator, engineer, researcher, writer, monitor, themis, kairos) that coordinate via shared QMD memory. Think of it as an operating system for your AI agents.

Do I need Hermes to use Caduceus?

Yes. Caduceus is a set of Hermes skills. It runs inside Hermes, not alongside it. If you don't have Hermes, install it from github.com/NousResearch/hermes-agent.

How is this different from Mission Control or Paperclip?

Mission Control and Paperclip are UI dashboards that wrap agent frameworks. Caduceus runs inside Hermes — it's not a separate dashboard. You talk to the orchestrator via Hermes chat, and it coordinates agents for you. No new UI to learn if you're already using Hermes.

Is Caduceus free?

The Terminal version is free and open source (MIT license). It runs entirely on your machine. Team (web dashboard, multi-user) and Hosted (managed service) are paid plans coming later. See pricing.

What does "runs on your machine" mean?

Caduceus installs into your existing Hermes setup at ~/.hermes/skills/. Your agents run on your hardware, using your API keys. Nothing is sent to our servers unless you sign up for Hosted (not available yet).

Can agents do dangerous things?

Caduceus has defense-in-depth: threat scanning on all prompts, draft-and-flag for irreversible actions (delete, drop, auth changes), bounded retries, and sub-agent depth limits. That said, agents can run arbitrary code on your machine — only run agents you trust, and review what they plan to do before approving.

What is QMD?

QMD stands for Query Markup Documents. It's a file-based knowledge base — markdown files with semantic search. In Caduceus, agents use QMD as a shared brain: they write their progress, findings, and learnings to QMD collections, and other agents read from it. It's human-editable, version-controllable, and searchable.

How is QMD different from a vector database?

Vector databases store embeddings for semantic search. QMD stores human-readable markdown that agents and humans can both read and edit directly. Caduceus uses QMD for coordination and institutional memory — not as a vector store. If you need dense vector search, you can add that separately.

Can I use Caduceus on Windows?

Not yet. The current version requires macOS or Linux. Windows support is planned for a future release.

How do I get help?

Open an issue on GitHub, or ask in the Hermes community Discord. For paid plans, priority support will be included.

When are Team and Hosted launching?

Team is planned for Q3 2026. Hosted is planned for Q4 2026. The Terminal version (free) is available now.
