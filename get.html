caduceus

Install

Requires Hermes Agent (macOS or Linux)

curl -fsSL https://raw.githubusercontent.com/mitchellbernstein/caduceus/main/scripts/install.sh | sh

This installs the Terminal version (free, open source).
Team and Hosted plans coming soon.

Alternatively, clone and run locally:

git clone https://github.com/mitchellbernstein/caduceus.git && cd caduceus && ./scripts/install.sh
